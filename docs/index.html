賽事背景

挑戰香港全部四條法定遠足徑

香港四徑超級挑戰賽（HK4TUC）要求選手連續完成麥理浩徑（100km）、 衛奕信徑（78km）、港島徑（50km）及大嶼山徑（70km），全程約 298km， 累計爬升逾 2 萬米，是香港最艱難的戶外賽事之一。四條徑之間需乘交通工具轉場。

本分析收集了 2021 至 2026 年六屆賽事的官方成績及 GPS 追蹤數據， 記錄了 50 名退賽選手的最後打卡檢查點，以了解選手在哪個賽段最易棄賽。

地圖一

各賽段退賽分佈

地圖以線條粗細及標注氣泡呈現各條徑的退賽熱度。 線條越粗代表在該賽段退賽的人數越多。氣泡內顯示合計人數， 並細分「途中退賽」與「完成後退賽」（即完成該段後在轉場期間放棄）。

⚠️ 此圖不顯示個人 GPS 座標，僅依官方檢查點打卡記錄統計， 避免因 GPS 漂移或轉場移動造成誤導。
資料來源：dottrack.asia 官方成績 · 路線來源：HK4TUC 官方賽道 KML · 統計期間：2021–2026
地圖二

退賽選手 GPS 最後位置

每個圓點代表一名退賽選手的 GPS 追蹤器最後記錄位置，按年份以顏色區分。 點擊圓點可查看選手姓名、國籍、退賽時間及完整追蹤連結。

⚠️ GPS 最後位置並不等同於退賽地點——部分選手的追蹤器在轉場期間仍在運作， 最後記錄位置可能為集合點或交通途中，與實際退賽位置有偏差。
GPS 數據來源：dottrack.asia 選手追蹤 · 路線來源：HK4TUC 官方賽道 KML · 統計期間：2021–2026
數據說明

方法論與數據局限

數據來源

  • 成績及檢查點數據：dottrack.asia Open Tracking 平台官方 API
  • GPS 追蹤數據：dottrack.asia 選手個人 KML 追蹤記錄
  • 官方賽道路線：HK4TUC 賽事方上傳之官方賽道 KML（22hk4tuc）
  • 覆蓋年份：2021、2022、2023、2024、2025、2026

重要說明

  • 「退賽位置」依據選手最後打卡的官方檢查點判斷，而非 GPS 座標
  • 「途中退賽」= 打卡進入某段後，未能到達下一個檢查點
  • 「完成後退賽」= 完成某段後，在轉場期間或下一段起點前放棄
  • 四條徑之間的轉場段（乘坐巴士/的士）不計入任何賽段退賽統計
  • 部分選手（8人）雖有 GPS 軌跡但無中間 CP 記錄，歸類為「麥理浩徑途中退賽」
  • 官方成績以 Retired 統一標記，本分析未區分主動退賽與強制退賽
